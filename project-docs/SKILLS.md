# SKILLS

## Capacidades necesarias para este proyecto
- Auditoria de themes Shopify
- Liquid y JSON templates
- Refactor controlado de Dawn
- UX para marketplace de servicios
- Arquitectura de contenido y conversion
- Preparacion para integraciones futuras

## Skills practicas del proyecto

### 1. Auditoria estructural
- Identificar personalizaciones frente a Dawn
- Separar contenido custom de logica base
- Detectar residuos de e-commerce clasico

### 2. Transformacion UX
- Pasar de tienda a marketplace de servicios
- Replantear homepage
- Replantear fichas de producto como servicios

### 3. Componentizacion reutilizable
- Reutilizar `card-product`, `collection-list`, `multicolumn`, `image-banner`
- Evitar duplicacion de secciones si una extension controlada basta

### 4. Trabajo seguro con aprobacion
- Cambios pequeños
- Revisión por bloques visibles
- Preparacion para rollback

### 5. Integracion futura
- Metacampos para servicios
- Modelado de talleres/proveedores
- Posible capa de booking
