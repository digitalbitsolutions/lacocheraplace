# Skill: Dawn Controlled Refactor

## Objetivo
Reutilizar Dawn inteligentemente sin romper la base del theme.

## Cuando usarla
- Al extender componentes existentes
- Al reemplazar bloques genericos por versiones mas alineadas al negocio
- Al decidir entre reutilizar o crear desde cero

## Enfoque
- Reutilizar `card-product`, `collection-list`, `multicolumn`, `image-banner`
- Evitar duplicacion de secciones si una adaptacion ligera resuelve
- Mantener el theme estable y mantenible

## Entregables
- Decision de reutilizacion o extension
- Lista de componentes afectados
- Riesgos de regresion
- Justificacion tecnica
