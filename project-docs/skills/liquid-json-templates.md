# Skill: Liquid + JSON Templates

## Objetivo
Modificar el theme con precision, usando la arquitectura nativa de Shopify.

## Cuando usarla
- Al editar `templates/*.json`
- Al tocar `sections/*.liquid`
- Al adaptar bloques, settings y configuraciones del editor

## Enfoque
- Cambios pequenos y reversibles
- Preferir configuracion JSON cuando baste
- Tocar Liquid solo cuando haya ganancia clara
- Mantener compatibilidad con el editor de temas

## Entregables
- Archivos editados
- Explicacion del impacto
- Supuestos y limitaciones
- Ruta de rollback
