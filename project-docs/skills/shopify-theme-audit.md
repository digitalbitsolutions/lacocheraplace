# Skill: Shopify Theme Audit

## Objetivo
Entender el estado real del theme antes de modificarlo.

## Cuando usarla
- Al inicio de una fase nueva
- Antes de tocar una plantilla importante
- Cuando haya dudas sobre que parte es custom y que parte es Dawn

## Enfoque
- Revisar `templates`, `sections`, `snippets`, `layout`, `config`
- Separar contenido personalizado de logica base
- Detectar residuos de e-commerce clasico
- Identificar componentes reutilizables

## Entregables
- Mapa de archivos clave
- Lista de personalizaciones existentes
- Riesgos de tocar cada zona
- Recomendacion del siguiente paso
