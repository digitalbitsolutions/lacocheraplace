# Skill: Marketplace UX Architecture

## Objetivo
Redefinir la experiencia para que el sitio se perciba como marketplace de servicios y no como tienda clasica.

## Cuando usarla
- Al trabajar homepage
- Al rediseñar fichas de producto/servicio
- Al definir CTAs, bloques y recorrido del usuario

## Enfoque
- Priorizar servicios, proveedores y reservas
- Reducir senales de catalogo tradicional
- Ordenar la home por intencion del usuario
- Diseñar flows claros: descubrir, comparar, solicitar, reservar

## Entregables
- Estructura recomendada por pagina
- Jerarquia de contenido
- CTAs por bloque
- Lista de elementos a eliminar, transformar o mantener
