# Skill: Versioning and Safe Rollout

## Objetivo
Proteger el proyecto con ramas, commits pequenos y rollback claro.

## Cuando usarla
- En cada iteracion
- Antes de tocar una zona sensible
- Antes de publicar o subir cambios a Shopify

## Enfoque
- Trabajar por ramas
- Commits pequenos y tematicos
- Revisar diff antes de avanzar
- No subir a produccion sin aprobacion

## Entregables
- Rama activa por fase
- Commit por lote visible
- Estado limpio del repo
- Estrategia de rollback
