# Skill: Shopify Preview and Publishing

## Objetivo
Pasar de cambios locales a una revision visual segura en Shopify.

## Cuando usarla
- Cuando un cambio local ya esta listo para ser validado
- Al preparar un theme borrador
- Antes de publicar algo aprobado

## Enfoque
- Preferir previews y themes no publicados
- Separar validacion visual de publicacion final
- Minimizar riesgo sobre el theme activo
- Usar MCP/CLI/conexion solo cuando haga falta

## Entregables
- Metodo de preview elegido
- Pasos de subida o sincronizacion
- Riesgos sobre tienda activa
- Criterio de aprobacion para publicar
