# Theme Audit

Use this reference when starting a new phase or touching an unfamiliar part of the theme.

## Audit order

1. Review `templates`
2. Review `sections`
3. Review `snippets`
4. Review `layout`
5. Review `config`

## What to identify

- What is custom content only
- What is custom logic
- What is still standard Dawn behavior
- What can be reused safely
- What still reinforces classic ecommerce patterns

## Output

- Key files
- Existing customizations
- Risks
- Next smallest useful change
